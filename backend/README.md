# Бэкендик
## Локальный запуск

### Подготовка
* Установить Erlang, Elixir и PostgreSQL
* Переименовать файл `.env.example` в `.env` и настроить его (указать данные от БД)

### Запуск сервера
* Выполнить `mix setup` (скачивает зависимости и настраивает базу данных)
* Запустить сервер командой `mix phx.server`
* Сервер будет доступен по адресу `http://localhost:4000`
